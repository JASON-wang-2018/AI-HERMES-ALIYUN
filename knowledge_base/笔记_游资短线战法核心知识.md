# A股游资短线战法核心知识笔记

> 来源：游资操盘体系整理 | 用途：超短交易学习与实战参考

---

## 一、游资操盘全流程：五阶段量价特征

### 1. 建仓（吸筹）阶段
- **量能特征**：温和放量，量能较前期均量放大1.5~2倍，股价小阳线缓慢推升
- **价格特征**：价格在相对低位区间波动，振幅较小（通常<10%），不轻易突破关键压力位
- **目的**：悄悄收集筹码，不引起市场注意，避免股价异动引发跟风
- **识别要点**：
  - 缩量回调时获得支撑（说明抛压轻）
  - 分时图出现"挤压式"吸筹（尾盘偷袭式拉升）
  - 盘口出现大单托底但不主动向上吃货

### 2. 试盘阶段
- **量能特征**：突然放量，量能是前期的2~4倍，次日又快速缩量
- **价格特征**：向上试盘突破关键压力位（如20日均线、前期高点），触碰后快速回落
- **目的**：测试上方抛压、跟风盘强度和市场情绪
- **识别要点**：
  - 长上影线（试盘K线）频繁出现
  - 试盘后缩量震荡整理2~5天
  - 若试盘后快速拉升突破则行情启动概率大

### 3. 洗盘阶段
- **量能特征**：量能萎缩至建仓期的0.5~0.7倍，呈现"量坑"
- **价格特征**：股价快速下跌砸出坑（幅度10%~20%），但整体维持区间震荡
- **目的**：清洗获利盘、提高市场平均持仓成本，减轻拉升阻力
- **识别要点**：
  - 下跌缩量、反弹放量
  - 关键均线（如5日、10日均线）获得支撑
  - 恐慌杀跌时出现大单承接（托单）
  - 出现"黄金坑"、"V形反转"等形态

### 4. 拉升阶段
- **量能特征**：量能持续放大，保持高换手（换手率通常>10%，强势股>15%）
- **价格特征**：股价以涨停或连续大阳线方式快速脱离成本区
- **目的**：实现利润最大化，吸引跟风资金接盘
- **识别要点**：
  - 涨停封板坚决，抛单稀少
  - 连续阳线实体饱满，影线短
  - 分时图呈45度角稳定爬升
  - 板块联动效应明显（龙头带动跟风股）

### 5. 出货阶段
- **量能特征**：量能异常放大（换手率>20%），但股价滞涨或创新高后快速回落
- **价格特征**：高位剧烈震荡，出现高位吊颈线、乌云盖顶等见顶K线组合
- **目的**：高位派发筹码，将账面利润兑现
- **识别要点**：
  - 涨停板反复打开（封板量减少）
  - 尾盘砸盘出货（14:30后快速杀跌）
  - 分时图出现"钓鱼线"（早盘冲高后单边回落）
  - 大幅高开低走，套牢追高资金

## 二、龙头连板股识别与买卖点

### 龙头股识别标准
| 指标 | 龙头股特征 |
|------|-----------|
| **板块地位** | 所属板块第一个涨停，股性最活，资金认可度最高 |
| **基本面** | 无重大利空，流通市值适中（20~100亿为宜） |
| **技术面** | 前期经过充分调整，筹码低位集中 |
| **情绪指标** | 涨停时封单量>流通市值的1%，换手率适中（8%~15%） |
| **联动性** | 能带动同板块跟风股跟涨，形成板块效应 |

### 买卖点判断

**买入选点：**
- **首板**：涨停一瞬间确定性买入（打板），或次日高开3%~5%回调时买入
- **二板**：确认连板意愿，次日竞价成交量>1000万股且高开买入
- **回调买**：连板后缩量回调至5日均线附近企稳买入（龙回头）

**卖出时点：**
- 持仓期间出现高位放量长阴线，次日无法反包 → 清仓
- 连续涨停后打开涨停板且15分钟内无法回封 → 减仓50%
- 股价跌破5日均线且反弹无力 → 清仓
- 板块整体退潮（跟风股批量跌停）→ 清仓

## 三、打板技巧（首板/二板/三板+）

### 首板（三板前）
- **买入时机**：涨停前10~30分钟内，股价出现连续大单扫货，逼近涨停时瞬间确认买入
- **选股标准**：
  - 流通市值<50亿（游资便于控盘）
  - 股性活跃（近1个月有涨停记录）
  - 处于板块启动初期或龙头地位
  - 分时图呈现"一波式"或"二波式"拉升
- **注意事项**：
  - 首板失败率较高，需严格止损
  - 避免高位首板（已涨50%+的首板风险大）
  - 避免缩量一字板后打板（无法确认换手）

### 二板（确认板）
- **买入时机**：次日竞价高开3%~7%，量比>3，且股价在开盘后15分钟内创出日内新高
- **核心逻辑**：二板是确认游资接力意愿的关键节点
- **二板类型**：
  - "一字板后换手二板"（高风险）
  - "实体阳线二板"（标准模式，最佳）
  - "跳空高开二板"（最强，确认主升浪）
- **注意事项**：二板若出现放量阴线（"烂板"），次日需观察开盘情绪再决定是否出局

### 三板及以上（妖股阶段）
- **买入时机**：连续缩量涨停后，T字板或换手板时买入（高位博弈）
- **三板信号**：
  - 场外踏空资金愿意高位接棒
  - 题材消息持续发酵
  - 交易所监管压力（停牌核查）前为最佳行情
- **卖出纪律**：
  - 三板后高位炸板（单日跌幅>8%），次日不反包即清仓
  - 连续涨停期间某日换手率>30%，警惕主力出货
  - 连续3个涨停后第四个交易日低开，视为走弱信号

## 四、-5%机械止损纪律

### 机械止损原则
> **核心铁律：单笔交易亏损不超过-5%，单日累计亏损不超过-8%，单月亏损不超过-20%。**

### -5%止损执行细则

| 场景 | 操作 |
|------|------|
| **建仓后立即下跌** | 跌破成本价-3%时预警，-5%无条件清仓，不抄底不补仓 |
| **盈利后回吐** | 从最高点回撤-5%时触发止盈止损（保本或微利出局） |
| **打板失败** | 涨停被砸开后2小时内无法回封，-5%止损 |
| **大盘系统性风险** | 大盘指数单日跌幅>2%，次交易日开盘即执行止损 |

### 止损执行要点
1. **不犹豫**：触价后5分钟内完成操作，不等待反弹
2. **不复盘**：止损后不对已止损股票进行"复盘买入"
3. **记录复盘**：每次止损后记录当日大盘情绪、个股逻辑是否成立
4. **拒绝拉均价**：绝对禁止通过补仓拉低成本价
5. **连续亏损处理**：连续3笔止损后暂停交易1个交易日，做情绪管理

## 五、超短选股标准

### 三大核心指标

#### 1. 换手率
| 换手率区间 | 市场含义 | 操作建议 |
|------------|----------|----------|
| <3% | 主力未进场或高度控盘 | 不参与 |
| 3%~8% | 正常活跃区间 | 可关注 |
| 8%~15% | 资金充分换手，活跃度高 | 重点关注 |
| >15% | 换手过于激烈，需区分是吸货还是出货 | 谨慎 |

#### 2. 量能（成交量）
- **量比**：开盘前30分钟量比>2.0，表示资金活跃
- **量能形态**：
  - "倍量过左峰"：突破前期高点时量能是左峰的2倍以上
  - "堆量"：股价上涨但量能温和放大，主力控盘拉升
  - "缩量上涨"：缩量创新高，主升浪特征（持筹待涨）
  - "放量滞涨"：量增价不涨，警惕出货

#### 3. 题材联动
- **联动强度**：龙头涨停后，3只以上跟风股涨幅>5%，视为强联动
- **题材等级**：
  - 一级题材（国家政策级别）：政策出台后持续炒作1~3个月（如AI、新能源）
  - 二级题材（行业利好）：事件驱动，持续3~7个交易日
  - 三级题材（个股传闻）：1~2日游资短线行情
- **选股优先级**：龙头>跟风最强股>板块补涨股

### 选股流程（五步法）
1. **盘前准备**：浏览当日财经新闻，锁定潜在炒作题材
2. **初选**：在同题材个股中筛选涨幅>5%且涨停时间较早的个股
3. **精选**：查看流通市值（20~100亿）、换手率（>8%）、股性（近1月有涨停）
4. **排雷**：排除ST、减持、解禁、近期大幅上涨的高位股
5. **确认**：临盘看分时图确认买点（突破均线/均线支撑有效）

## 六、亏钱原因与避坑指南

### 六大亏钱根源

#### 1. 逆势交易
- **表现**：大盘下跌时抄底、猜顶，持仓与大盘逆向
- **避坑**：大盘跌破20日均线进入空头趋势，只卖不买，降低仓位至30%以下

#### 2. 不止损（扛单）
- **表现**：亏损后不断补仓摊薄成本，最终深套
- **避坑**：严格遵守-5%止损纪律，亏损加仓是"死亡加速器"

#### 3. 追高打板
- **表现**：涨停板打开后追入，被套在高位（"天地板"）
- **避坑**：
  - 打板只在板块启动初期参与，避开高位接力
  - 炸板后观察2小时，无法回封即止损
  - 连续2个涨停后不再打板（盈亏比恶化）

#### 4. 频繁交易（过度交易）
- **表现**：每天都要买卖，空仓就难受，手续费侵蚀利润
- **避坑**：
  - 每周交易不超过5次（超短节奏）
  - 没有符合标准的机会就空仓等待
  - 手续费+印花税，单年交易成本可达本金的10%以上

#### 5. 缺乏交易系统（凭感觉）
- **表现**：没有选股标准、没有买卖点原则、情绪化交易
- **避坑**：
  - 制定并写下自己的交易规则（选股条件、买卖点、仓位管理、止损纪律）
  - 每日盘后复盘1小时
  - 用模拟盘或小资金验证交易系统至少3个月

#### 6. 盈亏同源认知偏差
- **表现**：赚了认为是技术好，亏了认为是市场问题
- **避坑**：
  - 每一笔交易记录买前预判、买后复盘
  - 统计胜率、单笔盈亏比、月度收益率
  - 持续跟踪交易数据，优化系统

### 十大避坑清单
1. ❌ 不炒ST、亏损股、负债率>80%的个股
2. ❌ 不在涨停板上排一字板（无法确认换手）
3. ❌ 不在14:30后打板（次日溢价不确定）
4. ❌ 不买大盘股（>300亿流通市值）超短
5. ❌ 不在跌停板上"捡便宜"
6. ❌ 不重仓单只股（单票仓位≤30%）
7. ❌ 不在市场情绪退潮期（昨日涨停股今日跌停率>10%）强行交易
8. ❌ 不相信"内幕消息"、"老师带盘"
9. ❌ 不在连续亏损后加大资金试图翻本
10. ❌ 不在持仓股出现重大利空后心存侥幸不止损

## 附：超短交易纪律卡（背诵版）

```
一、选股：换手>8%、量比>2、题材联动强、涨停时间早
二、买点：突破均线/回踩均线企稳/二波拉升启动点
三、止损：-5%机械止损，不抄底不补仓
四、卖出：高位炸板不回封→减仓；跌破5日线→清仓
五、仓位：单票≤30%，不满仓，留30%现金
六、节奏：强势市场8成仓，弱势市场3成仓，空头市场空仓
七、心态：连续3笔亏损→停止交易1天
八、记录：每笔交易记录预判与结果，每月总结复盘
```
