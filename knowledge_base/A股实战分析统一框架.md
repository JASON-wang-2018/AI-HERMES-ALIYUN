## 第一步：定性定位——庄家阶段识别

> 一切分析的前提：先判断这只股票现在"庄家在哪一步"。

### 庄家生命周期8阶段（曹明成体系）

| 阶段 | 核心任务 | 均线特征 | 量价特征 | 操作价值 |
|------|---------|---------|---------|---------|
| **1 建仓** | 压价收集 | 底部横盘 | 极度缩量 | ⭐⭐ 布局期 |
| **2 吸筹** | 低位整固 | 缓慢上移 | 温和放量 | ⭐⭐⭐ 低吸期 |
| **3 试盘** | 测抛压 | 脉冲式异动 | 突然放量 | ⭐⭐ 识别期 |
| **4 洗盘** | 清浮筹 | 回踩不破MA | 缩量回调 | ⭐⭐⭐⭐ 机会期 |
| **5 拉升** | 脱离成本 | 多头排列 | 量增价涨 | ⭐⭐⭐⭐⭐ 主升期 |
| **6 砸盘** | 掩护出货 | 高位震荡 | 放量滞涨 | ⭐ 离场期 |
| **7 出货** | 高位派发 | 均线死叉 | 天量长阴 | ⭐⭐⭐⭐⭐ 派发期 |
| **8 砸盘退出** | 清仓离场 | 破位下跌 | 恐慌放量 | ⭐ 观望期 |

### 定位判断树

```
① 位置判断：
   历史位置 < 35% → 考虑 建仓/吸筹/洗盘
   历史位置 35~65% → 考虑 拉升/砸盘/出货
   历史位置 > 65% → 重点警惕出货

② 均线结构：
   MA5 > MA10 > MA20 > MA60 → 多头（拉升 or 洗盘未破）
   MA5 < MA10 < MA20 < MA60 → 空头（建仓前 or 出清后）
   均线缠绕 → 震荡（洗盘 or 派发途中）

③ 趋势方向：
   近期高点到低点跌幅 < 20% → 可能洗盘
   近期高点到低点跌幅 > 30% → 可能派发后砸盘
   近期低点涨超 30%+ → 可能有庄家拉升

④ 量价关系（最关键）：
   下跌缩量 + 上涨放量 → 吸筹/洗盘（主力在场）
   上涨缩量 + 下跌放量 → 派发/出货（主力在跑）
```

### 量化输出

```
【阶段定位】：______阶段（建仓/吸筹/洗盘/拉升/派发/出清）
【置信度】：高/中/低（多维验证有多少信号支持）
【关键证据】：3条最有力的证据
【最大疑点】：1个与判断矛盾的信号（必须解释）
```

## 第二步：多维验证——9维度证据收集

> 定位之后，用9个维度验证判断。维度越多验证，结论越可靠。

### 维度1：均线系统（20%权重）

**检查项：**
- MA5/MA10/MA20/MA60 多头 or 空头 or 缠绕
- 收盘价与均线的位置关系
- 均线斜率方向（最近5日）
- 均线是否作为支撑/压力发挥作用

**量化信号：**
```
均线多头信号：收盘>MA5>MA10>MA20 且 均线向上发散
均线空头信号：收盘<MA5<MA10<MA20 且 均线向下发散
回踩MA确认：回踩MA5或MA10时量能萎缩+次日收复
```

### 维度2：K线形态（15%权重）

**关键形态识别：**

| 形态 | 出现在低位 | 出现在高位 | 信号强度 |
|------|-----------|-----------|---------|
| 锤子线（长下影） | 底部吸筹 | 顶部是吊颈 | 强 |
| 光头光脚阳 | 启动信号 | 可能是拉升尾声 | 中 |
| 十字星 | 多空平衡 | 高位是反转 | 强 |
| 吞没线（大阳包阴） | 底部反转 | 顶部是反转 | 强 |
| 乌云盖顶 | — | 高位派发 | 强 |
| 早晨之星 | 底部信号 | — | 强 |

**试盘线识别（庄家特有）：**
- 长上影线（冲高回落）+ 缩量 = 试盘（测试抛压）
- 次日能收复 = 洗盘；不能收复 = 派发

### 维度3：量价关系（25%权重，最核心）

**六种量价关系：**

```
1. 价涨量增 + 价跌量缩
   → 健康上涨，主力在推升
   → 评分 +8

2. 价涨量缩
   → 量价背离，可能是反弹尾声
   → 评分 -5

3. 价跌量增（不是天量）
   → 可能是吸筹（打压吸筹）
   → 需要结合位置判断

4. 价跌量增（天量）
   → 恐慌盘出逃，主力在接
   → 底部天量是积极信号

5. 高位放天量 + 滞涨
   → 主力出货（明确离场信号）
   → 评分 -10

6. 长期缩量后突然放量
   → 变盘前兆，结合方向判断
```

**近5日量价结构评分：**
- 量价同增：+6
- 量价同减：+3
- 价涨量缩：-5
- 价跌量增（非恐慌）：+2
- 价跌量增（恐慌）：+4（低位）

### 维度4：MACD指标（15%权重）

**核心信号：**
```
零轴上方金叉 → 短多（拉升 or 洗盘结束）
零轴下方死叉 → 短空（出货 or 砸盘）
零轴上方死叉 → 高位预警（可能出货）
零轴下方金叉 → 低位积累（可能吸筹结束）
MACD背离：
  价格创新低 + MACD没有创新低 → 底背离（底部信号）
  价格创新高 + MACD没有创新高 → 顶背离（顶部信号）
```

**庄家行为对应：**
- MACD在0轴附近反复金叉死叉 → 吸筹/洗盘（反复震荡）
- MACD稳定在0轴上方 → 拉升期（趋势健康）
- MACD在0轴上方红柱持续萎缩 → 主力在减仓

### 维度5：KDJ摆动指标（10%权重）

```
K < 30 → 超卖（低位金叉是买入机会）
K > 80 → 超买（高位死叉是卖出信号）
K从下穿越D（金叉）+ D在低位 → 较好买入信号
KDJ高位钝化 → 强势股（不要轻易看空）
KDJ底背离 → 底部信号
```

### 维度6：布林带（10%权重）

```
股价 > 上轨 → 超买（高位预警，尤其是+Boll见顶）
股价 < 下轨 → 超卖（低位机会，尤其是-Boll见底）
股价在中轨附近运行 → 震荡整理（方向不明）
布林带收口（波动率降低）→ 变盘前兆（即将选择方向）
```

**配合MACD使用：**
- 布林下轨 + MACD底背离 → 较强底部信号
- 布林上轨 + MACD顶背离 → 较强顶部信号

### 维度7：板块与题材（5%权重）

```
板块效应：
  所属板块当日涨停家数多 → 板块主线（加分）
  所属板块当日跌幅大 → 拖累个股（减分）
  个股是板块龙头 → 溢价更高
  个股是跟风股 → 弹性更弱

题材催化剂：
  有政策/业绩/并购等明确催化剂 → 加分
  无题材支撑，纯技术走势 → 谨慎
```

### 维度8：ATR波动率（5%权重）

```
ATR > 3% → 高波动（趋势强 or 风险大）
ATR < 1.5% → 低波动（主力控盘/盘整）
ATR急剧放大 → 变盘信号（可能突破）
ATR在高位 → 警惕假突破（多头陷阱）
```

### 维度9：RSI相对强弱（5%权重）

```
RSI < 30 → 超卖区（低位金叉是机会）
RSI > 70 → 超买区（高位死叉是风险）
RSI顶背离 → 主力在减仓
RSI底背离 → 蓄势反弹
RSI在50以上 → 偏强；50以下 → 偏弱
```

## 第四步：走势推演——三种路径概率化

### 路径一：主升延续

```
触发条件：放量中阳突破近20日高点 + MACD金叉 + 板块主线
概率：30~45%（牛市/主线明确时上调）
目标位：前高 + 15~25%
止损：MA5跌破离场
```

### 路径二：震荡分化

```
触发条件：在关键支撑和压力之间反复
概率：35~45%（震荡市默认路径）
操作：高抛低吸，快进快出，不恋战
关键：高抛点（压力位）/ 低吸点（支撑位）
```

### 路径三：破位下跌

```
触发条件：跌破近20日低点 + 量能放大
概率：20~30%（节前/高位时上调）
目标位：前低下方 10~15%
止损：必须纪律性离场
```

### 止损三级体系

```
止损1（短线）：MA5跌破离场
止损2（波段）：MA10跌破离场
```

```
首止盈：成本+10% 减仓1/3
次止盈：成本+20% 再减1/3
底仓：趋势不破不走，让利润奔跑
```

### 仓位管理原则

```
总仓位 = 市场确定性 × 标的确定性
主升市：60~80%
震荡市：20~40%
不明朗：0~20%

单票上限：总资金20%
加仓原则：确认后再加，不盲目补仓
```

## 第七步：基本面与消息面验证

> 基本面是战略层，技术面是战术层。基本面决定持股信心，消息面影响短期波动。

### 基本面三问

```
① 业绩趋势：最近1-2个季度业绩是否边际改善？
② 行业地位：龙头/跟风/边缘？
③ 估值水位：PE/PB在历史什么分位？
```

### 消息面下的庄家行为

**借大势洗盘**
- 时机：大盘下跌时，个股顺势砸盘
- 识别：大盘跌，个股缩量小跌；大盘稳，个股立即反弹
- 买点：大盘分时企稳时，个股不破前低

**借利好洗盘**
- 时机：利好公布前，庄家借机洗盘
- 识别：利好前连续小阴线打压，利好公布当日低开高走
- 买点：利好次日缩量回调不破当日开盘价

**借利空洗盘**
- 时机：利空出来后，庄家借机砸盘
- 识别：利空出来，股价跌不动（低开高走或缩量不跌）
- 买点：利空当日尾盘或次日开盘不创新低

**借利好出货（多头陷阱）**
- 形态：利好公布→跳空高开→高位震荡→出货
- 识别：跳空幅度大（≥5%）+ 高位爆量（换手≥10%）+ 封板后炸板
- 卖点：利好次日无法连板 / 高开低走

## 附一：洗盘买点体系（5大买点）

> 洗盘是庄家拉升前最后一次洗浮筹，**缩量回调是买点**。位置决定性质——低位缩量回调=洗盘，高位放量滞涨=出货。

### 买点1：打压式洗盘买点
- **形态**：庄家故意砸盘，连续中大阴线制造恐慌
- **量化条件**：
  - 成交量：打压时缩量（<均量50%）
  - 回调位置：不破重要支撑（MA60/前低/筹码密集区）
  - 打压幅度：10~25%（凶狠型30%+）
- **买点**：缩量企稳时（锤子线/刺透形态当天收盘或次日）
- **止损**：再创新低且1-3日内无法收复

### 买点2：横盘筑平台洗盘买点
- **形态**：窄幅震荡，波动≤10%，持续2~8周
- **量化条件**：
  - 成交量：地量（主力控盘锁筹）
  - 振幅：越小越好（≤8%）
  - 方向：最终向上突破
- **买点**：放量突破平台最高点时（真突破），量≥前5日均量1.5倍
- **止损**：跌破平台下沿3%

### 买点3：边拉边洗买点（最常见！）
- **形态**：拉升过程中边涨边洗，K线"进二退一"
- **量化条件**：
  - 量能：拉升放量 / 回调缩量（健康节奏）
  - 均线：回调不破MA5/MA10
  - 调整幅度：一般≤15%
- **买点**：回调缩量企稳在MA5/MA10附近，次日翻红跟进
- **止损**：跌破MA10且次日无法收复

### 买点4：T字板买点
- **形态**：涨停打开又封回，K线呈"T"字
- **量化条件**：
  - 封板时间：早盘10:30前回封强
  - 成交量：当日量能≤前日2倍
  - 换手率：≤15%（过高=主力在出货）
- **买点**：回封瞬间，或次日低开确认不破昨日收盘时
- **止损**：次日低开低走且跌破开盘价

### 买点5：冲高回落式洗盘买点
- **形态**：当日冲高后大幅回落，K线带上影线
- **量化条件**：
  - 回落幅度：≥5%回落才有意义
  - 量能：冲高放量+回落缩量
  - **位置是关键**：低位=洗盘，高位=出货
- **买点**：次日缩量不破前日低点，或连续3日企稳
- **止损**：跌破前日实体中点

## 附二：出货卖点体系（7大模式）

> 识别主力跑了！以下任意2个信号同时出现 → 警惕；3个以上 → 必须离场。

### 卖点1：高位横盘出货
- **形态**：股价在高位（≥80%涨幅）震荡，阴阳交错，振幅15~20%
- **量化条件**：
  - 成交量：放一段缩一段（主力间歇出）
  - 均线：逐渐走平（多头变空头）
  - 时间：高位横盘超过3周=危险信号
- **卖出信号**：
  - 连续3日不创新高
  - 跌破MA10且次日无法收复
  - 连续堆量滞涨

### 卖点2：拉高出货（最常见）
- **形态**：分时图早盘快速拉升→高位横盘→尾盘小幅跳水
- **量化条件**：
  - 拉升速度：越快越危险（直线急拉=诱多）
  - 封板后反复开板=主力减仓
  - 换手率：≥20%=高位区
- **卖出信号**：
  - 一口气拉到8%+量能跟不上（背离）
  - 封板后反复打开，量能堆积
  - 尾盘不开板但量能持续放大

### 卖点3：震荡式出货
- **形态**：反复冲高→回落，再冲高→再回落，振幅20~30%
- **量化条件**：
  - 高点一次比一次低
  - 低点逐渐下移
  - 量能：反弹缩量，下跌放量
- **卖出信号**：每次反弹到前高附近都是减仓点

### 卖点4：打压式出货（最凶狠）
- **形态**：庄家直接砸盘，股价快速暴跌
- **量化条件**：
  - 跌停或连续大跌
  - 跌停打开当日巨量换手（主力对倒）
  - 利空配合
- **卖出信号**：跌停打开第一波反弹是最后逃命机会，不要再买入！

### 卖点5：涨停板出货（近年最流行！）
- **形态**：封涨停板→大量出货→尾盘炸板→次日低开
- **量化条件**：
  - 封板时间：越晚越危险（14:30后封→诱多概率大）
  - 封单量：<1000万且快速撤单=主力在跑
  - 炸板幅度：≥5%=危险
  - 次日：低开低走跌幅≥5%
- **卖出信号**：
  - 涨停板打开第一时间减仓
  - 次日低开反弹至前日收盘价附近卖出
- **口诀**："涨停板封单薄，次日低开跑不迟"

### 卖点6：整数位关口出货
- **形态**：在整数关口（10/20/30/50等）反复震荡不过
- **量化条件**：
  - 到达整数位：多次上攻但不过
  - 量能：到关口就放量，不过去
  - 均线：逐渐转空
- **卖出信号**：关口3次不过+量能背离

### 卖点7：对倒/M头/台阶式出货
- **对倒出货**：量能异常放大但价格变化不大
- **M头出货**：双顶结构，第二个头顶量能萎缩
- **台阶式出货**：每涨一段横盘出货，然后继续拉再出货

## 附三：K线组合卖点（识别顶部信号）

### 避雷针顶风声急
- **形态**：上影线很长（≥实体2倍）
- **位置**：高位（涨幅≥50%）出现=强烈卖出信号
- **变体**：十字星 + T字线 + 倒锤子线都是同类
- **操作**：次日无法收复上影线最高点=卖出

### 倾盆大雨快收衣
- **形态**：大阳线后，次日高开低走直接覆盖前日阳线
- **操作**：次日收盘低于前日阳线实体中点=卖出

### 乌鸦群飞势转淡（三只乌鸦）
- **形态**：连续3根及以上小阴线
- **操作**：高位出现+量能放大=卖出

### 黄昏之星日已西
- **形态**：阳线→十字星/小实体→大阴线（3日结构）
- **识别**：中间星线缩量=洗盘，爆量=出货
- **操作**：第三根阴线收盘低于星线最低点=卖出

### 死亡穿越：一阴穿三线
- **形态**：一根大阴线同时跌破MA5/MA10/MA20
- **操作**：次日无法收复任何一根均线=清仓

## 附四：洗盘vs出货终极分辨（最重要！）

> 同样的K线，在低位=洗盘（可以买），在高位=出货（必须跑）。位置是核心！

### 六维分辨表

| 特征 | 洗盘（主力在） | 出货（主力跑了） |
|------|--------------|----------------|
| **位置** | 涨幅≤30%，相对低位 | 涨幅≥80%，高位 |
| **量能** | 下跌缩量 | 下跌放量 |
| **均线** | 不破重要均线 | 均线逐渐转空 |
| **次日** | 多能收复 | 无法收复 |
| **时间** | 短（1~2周） | 长（3周以上） |
| **目的** | 洗浮筹 | 减仓 |

### 震荡分辨

| 特征 | 震荡出货 | 震荡洗盘 |
|------|---------|---------|
| 高点 | 越来越低 | 越来越高（上升趋势） |
| 低点 | 逐渐下移 | 越来越高 |
| 量能 | 反弹缩量，下跌放量 | 回调缩量（主力没跑） |
| 振幅 | 越来越大 | 越来越小 |

### 利好分辨

| 特征 | 利好出货（多头陷阱） | 利好洗盘 |
|------|-------------------|---------|
| 公布当日 | 跳空高开→快速涨停→封板打开 | 低开高走/封板稳健 |
| 换手率 | ≥15%=危险 | ≤10%=健康 |
| 之后走势 | 高位震荡无法再涨 | 缩量回调后继续涨 |

### 横盘分辨

| 特征 | 横盘出货 | 横盘洗盘 |
|------|---------|---------|
| 位置 | 高位（涨幅≥80%） | 低位（涨幅≤30%） |
| 量能 | 时大时小（主力间歇出） | 持续地量（主力锁筹） |
| 破位时 | 放大量 | 缩量 |

### 打压分辨

| 特征 | 打压出货 | 打压洗盘 |
|------|---------|---------|
| 走势 | 快速暴跌→跌停→无量（跑不掉） | 砸盘但有底线 |
| 量能 | 跌停打开巨量换手（主力疯狂出） | 缩量（主力没跑） |
| 次日 | 继续低开低走 | 快速收复 |

## 附五：反转形态买卖点

### 双底（W底）/ 双顶（M顶）

**W底（底部反转，买点）**
- 形态：两个相近低点，第二个更高
- 颈线：两底之间的反弹高点连线
- 买点：突破颈线时（放量1.5倍），或回踩颈线不破时
- 量能：第二个底量<第一个底（底背离）

**M顶（顶部反转，卖点）**
- 形态：两个相近高点，第二次更低
- 颈线：两峰之间的低点连线
- 卖点：跌破颈线时（颈线下方3%确认）
- 量能：第二次顶量<第一次（顶背离）

### 头肩顶 / 头肩底

**头肩顶（最可靠顶部形态）**
- 形态：左肩↑→头（更高）→右肩（更低）
- 颈线：左肩和头之间的低点连线
- 卖点：跌破颈线时
- 量能：右肩量<左肩量（确认）

**头肩底（最可靠底部形态）**
- 形态：左肩↓→头（更低）→右肩（更高）
- 买点：突破颈线时（放量确认）

---

## 附六：整理形态买卖点

### 上升旗形（最常见中继，看涨）
- 形态：陡峭上升后，小幅回调形成旗面
- **买点**：突破旗形上轨时，量≥前5日均量1.5倍
- 目标位：旗杆高度×1+旗面突破点
- **止损**：跌破旗面下轨3%

### 箱形整理
- 形态：股价在固定区间（±5~10%）来回震荡
- **操作**：
  - 区间上沿：缩量不过=减仓点
  - 区间下沿：缩量企稳=轻仓试买点
  - **突破确认**：放量≥2倍均量突破上沿时跟进
  - **止损**：收盘跌破箱底3%

### 三角形整理
- **上升三角形**（看涨）：高处平，低处越高，向上突破买
- **下降三角形**（看跌）：低处平，高处越低，向下突破卖

## 附七：空头陷阱（假跌破，买点）

**空头陷阱 = 庄家故意跌破支撑，吓出散户，然后V反拉升**

### 识别三要素

```
1. 跌破后快速拉回（1~3日内）
2. 跌破时缩量（主力没跑）
3. 拉回时放量（主力入场）
```

### 买点

```
放量阳包阴吞没前一日阴线时 = 买点
止损：再创新低且无力收复
```

---

## 附八：多头陷阱（假突破，不要买！）

**多头陷阱 = 主力故意向上突破，诱多接盘，然后反手下杀**

### 识别要点

```
- 突破幅度小（≤3%）
- 突破后1~3日内无力站稳
- 突破时量能不足（缩量突破）
- 借利好突破但封板后炸板
```

### 对策

```
等突破后回踩确认再买
高位金叉不追，等死叉确认
大盘红，个股不跟=减仓信号
```

## 附九：分时图卖点（当日实时决策）

### 高开探天出货定式
- **形态**：竞价高开（≥5%）→快速冲高→高位震荡→大幅回落
- **特征**：高开幅度越大越危险；冲高时无量=诱多
- **卖点**：跌破分时均价线时；或冲高后回落跌破开盘价

### 高台跳水出货定式
- **形态**：股价在高位横盘→突然快速杀跌→跌停
- **卖点**：跌停打开第一时间减仓

### 刀刃波出货定式（最凶险）
- **形态**：分时图呈锯齿状快拉快砸，股价上蹿下跳
- **含义**：主力高度控盘+疯狂出货
- **卖点**：出现刀刃波=立即卖出，不要等

### 尾盘拉升出货定式
- **形态**：14:00后快速拉升→试图封板→封不住→回落
- **特征**：越晚封板越危险；次日多低开
- **卖点**：尾盘无法封板+量能堆积=卖出

## 附十：短线黑马卖点

### 逼空连板式黑马
- **卖点**：
  - 爆量开板第一波=减仓点（不是卖完）
  - 开板后回封失败=清仓点
  - 回调到MA5不破=可买回（做第二波）
- **持仓原则**：持仓成本线上不轻易卖，MA5不破不出

### 连板后低走类黑马
- **形态**：连板后高位长阴杀跌
- **卖点**：长阴次日无法反包=清仓；低开后反弹到长阴半分位=卖出

### 超短线黑马
- **买点**：当日涨停前排买入（14:30前封板）
- **卖点**：
  - 次日无法连板=卖出
  - 次日低开-5%以下=止损
  - 次日缩量反包=可留

## 附：曹明成买卖点口诀（记忆版）

### 买点口诀
```
1. "缩量回调是金" — 拉升后缩量回调不破均线=最佳买点
2. "横有多长竖有多高" — 横盘整理越久，未来涨幅越大
3. "平台突破要跟进" — 放量突破整理平台颈线=确认买点
4. "地量地价见底" — 量能极度萎缩=主力吸筹末期=中线买点
5. "金叉缩量是洗盘" — MACD在0轴上方金叉+缩量=短线买点
6. "回踩不破是机会" — 突破后回踩颈线/均线不破=稳健买点
7. "买在缩量、整理、蓄势时"
```

### 卖点口诀
```
1. "放量滞涨要小心" — 上涨放大量但涨幅收窄=主力减仓
2. "高位横盘是出货" — 涨幅≥80%后高位横盘=卖出信号
3. "跌破均线要离场" — 股价跌破MA20且无力收复=止损
4. "利好兑现是利空" — 利好公布后跳空高开低走=卖出
5. "死亡穿越快跑" — 一阴穿三线=主力不计成本出
6. "天量见天价" — 放出历史天量=阶段性顶部确认
7. "卖在放量、拉升、众人追时"
```

## 附：交易心理控制清单

### 每笔交易前检查

```
□ 这只股票现在在哪个庄家阶段？
□ 支持这个判断的证据有3条以上吗？
□ 最大的疑点是什么？能解释吗？
□ 止损位在哪里？是否接受？
□ 这笔交易符合我的体系吗？
□ 是情绪驱动还是逻辑驱动？
□ 仓位是否合理（单票≤20%）？
□ 板块和题材是否支持？
□ 洗盘vs出货：位置/量能/均线/次日表现 四维都验证了吗？
□ 是洗盘买点还是出货陷阱？
```

### 交易后复盘

```
□ 是否执行了计划？
□ 止损设置合理吗？
□ 有没有情绪化交易？
□ 如果亏损，原因是体系内还是体系外？
□ 如果盈利，是运气还是实力？
□ 下次如何改进？
□ 有没有识别出庄家陷阱（多头/空头）？
□ 买点是洗盘还是出货陷阱？
```
