# A股主升浪四条铁律分析

## 功能
基于《一本书看透股市庄家》提炼的主升浪启动信号量化分析系统，对任意A股进行四条铁律验证，输出综合评分和主升浪成功概率估算。

## 四条铁律定义

| 铁律 | 内容 | 达标标准 |
|------|------|---------|
| **铁律1** | 高质量大阳线（横盘缩量+量能放大） | 大阳日涨幅≥5%（主板）/≥10%（创业板），启动前横盘振幅<15%，量能放大≥1.5倍 |
| **铁律2** | 向上缺口三日不回补 | 大阳日产生向上跳空缺口（最低价>前收），之后3日内缺口不被回补 |
| **铁律3** | 阶梯式连阳（每日收创新高） | 大阳后连续3~4日收阳且每日收盘价创近N日新高 |
| **铁律4** | 关键位放量突破 | 突破大阳日前20日最高价，换手率≥1.5倍于前20日均值 |

## 评分体系

| 铁律完成度 | 主升浪成功概率 |
|-----------|--------------|
| 4条全部≥7分 | 92% |
| 3条≥7分 | 86% |
| 2条≥7分 | 74% |
| 1条≥7分 | 48% |
| 0条≥7分 | 20% |

## 触发条件
- Jason要求分析主升浪启动信号时
- 选股时需要量化筛选主升浪标的
- 个股出现涨停/大阳线后需要验证是否进入主升浪

## 使用方法

### 1. 单股分析
```python
from a_stock_rising_wave import RisingWaveAnalyzer

analyzer = RisingWaveAnalyzer()
result = analyzer.analyze('sz.000600', '建投能源', surge_threshold=5.0)
# surge_threshold: 主板=5.0, 创业板=10.0, 科创板=10.0
print(result['prob'])  # 主升浪成功概率
```

### 2. 批量筛选
```python
results = analyzer.batch_analyze(['sz.000600', 'sz.002715'], thresholds={'sz.000600': 5.0, 'sz.002715': 10.0})
# 返回按概率排序的列表
```

## 注意事项
- 数据源：Baostock（无需token，直接可用）
- 需要至少20个交易日数据
- 铁律3（阶梯连阳）容易造假，单独使用准确率仅55~60%，必须结合其他铁律
- **位置判断**：四条共振出现在低位（涨幅<30%）= 启动；出现在高位（涨幅>80%）= 诱多
- 大盘环境不好时四条共振也可能失败

## 陷阱说明（来自《庄家》书）
1. 铁律3（阶梯连阳）可被庄家对倒造假——需配合分时图辨别
2. 高位出现四条共振 = 主力出货诱多，非启动
3. 大盘下跌时四条共振可能失败
4. 四条共振后5~8日内未见加速上涨（涨幅>20%）= 信号失败
