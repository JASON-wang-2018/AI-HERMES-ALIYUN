## 一、核心分析步骤

### 第1步：定性定位——趋势与生命周期判断

**目标**：先判断"这只股票现在处在哪一阶段"，避免在错误周期内做精细分析。

**核心判断维度**：
- **均线结构**
  - 多头：5/10/20/60 向上发散
  - 空头：均线死叉、下压
  - 震荡：均线缠绕
- **股价位置**
  - 历史/阶段低位 → 吸筹可能
  - 中位 → 换手、洗盘
  - 高位 → 派发风险
- **趋势属性**：主升 / 震荡 / 下跌反弹 / 出清末期

**输出**：股票所处生命周期阶段 + 是否值得进入"精细分析阶段"

---

### 第2步：多维分析——证据收集而非单点判断

#### 1️⃣ 趋势与均线
- 均线是否被有效回踩而不破
- 回踩是否缩量
- 均线斜率是否保持上行

#### 2️⃣ K线形态与组合
- **试盘线**：长上影、缩量
- **洗盘线**：假阴真洗
- **启动线**：放量中阳/长阳
- 关键位置是否出现连续K线确认

#### 3️⃣ 量价关系（核心）
- 上涨是否放量、回调是否缩量
- 放量但价格不动 → 吸筹/对倒
- 放量长阴但不破位 → 洗盘概率

#### 4️⃣ 分价表/成交结构
- 成交密集区是否上移
- 是否存在明显"锁仓成本区"
- 高位成交是否快速被吃掉

#### 5️⃣ 动能与节奏
- 连续K线的"攻击—回撤—再攻击"结构
- 是否符合主力常见操盘节奏，而非散户情绪波动

#### 6️⃣ 板块与题材环境
- 所属板块是否当日/近期活跃
- 是否有政策、事件或资金逻辑
- 个股是否为板块龙头/次龙/补涨/跟风

#### 7️⃣ 主力资金行为（定性）
- 资金介入是"推升"还是"托而不攻"
- 是否存在"护盘行为"
- 是否刻意制造恐慌但价格不破结构

---

### 第3步：综合评分——量化主观判断（100分制）

| 维度 | 权重 |
|------|------|
| 趋势结构 | 20分 |
| 量价健康度 | 25分 |
| K线信号质量 | 15分 |
| 主力行为可信度 | 20分 |
| 板块与情绪环境 | 20分 |

**评分标准**：
- 80+：强势主升结构
- 60–80：可操作，但需择时
- 60以下：观望或仅适合短线博弈

---

### 第4步：走势推演——概率化而非预测

**三种路径**：
1. **主升延续（高概率）**：缩量回踩不破关键均线
2. **震荡洗盘（中概率）**：放量不涨、反复假突破
3. **趋势破坏（低概率但必须防）**：放量跌破关键结构位

---

### 第5步：制定策略——按持仓状态区分

| 持仓状态 | 策略 |
|----------|------|
| 空仓者 | 等确认信号（放量突破/回踩确认） |
| 轻仓者 | 回踩关键位可加仓 |
| 重仓者 | 风控第一，不加仓，只做减法或持有 |

---

### 第6步：设定风控——系统性活下来

- **止损位**：结构位/均线位/成本密集区下沿
- **止盈策略**：分批止盈，而非一次性
- **观察点**：放量不涨 / 板块退潮 / 主力护盘消失

---

### 第7步：基本面——识别真正有价值的公司

## 三、K线组合形态速查

| 形态名称 | 特征 | 信号意义 |
|----------|------|---------|
| 底部吞没 | 下跌后出现大阳吞没前日阴线 | 看涨 |
| 顶部吞没 | 上涨后出现大阴吞没前日阳线 | 看跌 |
| 红三兵 | 连续三根上涨阳线，实体渐大 | 看涨 |
| 黑三鸦 | 连续三根下跌阴线 | 看跌 |
| 锤子线 | 下影线长于实体2倍+，出现在低位 | 底部信号 |
| 吊颈线 | 同形态出现在高位 | 顶部信号 |
| 曙光初现 | 大跌后出现低开高走阳线 | 底部信号 |
| 乌云盖顶 | 上涨后出现高开低走阴线 | 顶部信号 |
| 早晨之星 | 三根K线组合：跌→星→涨 | 底部信号 |
| 黄昏之星 | 三根K线组合：涨→星→跌 | 顶部信号 |
