## 一、基础知识层

### 1.1 K线理论
- **单根K线**：光头光脚、十字星、锤子线、吊颈线等
- **K线组合**：吞没、孕线、乌云盖顶、红三兵、黑三鸦等
- **K线位置**：顶部、底部、趋势中
- **成交量配合**：量增、量缩、量价背离

### 1.2 基本面分析
- 财务报表：利润表、资产负债表、现金流量表
- 估值指标：PE、PB、PS、EV/EBITDA
- 成长性：营收增速、净利润增速、毛利率
- 行业地位：市场份额、护城河、竞争格局
- 宏观周期：GDP、CPI、PPI、货币政策

### 1.3 量价关系
- 量增价涨：资金入场确认
- 量增价跌：主力出货或恐慌抛售
- 价升量缩：趋势可能反转
- 地量见地价：底部信号
- 天量见天价：顶部信号

## 三、主力行为层

### 3.1 坐庄流程
```
建仓 → 吸筹 → 试盘 → 洗盘 → 拉升 → 砸盘 → 出货
```
- **建仓**：压低股价慢慢买
- **吸筹**：低位收集带血筹码
- **试盘**：测试抛压/跟风
- **洗盘**：洗出获利盘
- **拉升**：快速脱离成本区
- **砸盘**：高位出货或低位吓出恐慌盘
- **出货**：高位派发

### 3.2 资金流向
- 主力净流入/流出
- 超大单、大单、中单、小单
- 龙虎榜数据

## 四、市场认知层

### 4.1 市场周期
- 牛市：初期的犹豫 → 中期的爆发 → 末期的疯狂
- 熊市：初期的绝望 → 中期的反弹 → 末期的麻木
- 震荡市：板块轮动、热点切换

### 4.2 板块轮动
- 券商 → 科技 → 消费 → 周期 → 金融
- 政策驱动型轮动
- 资金驱动型轮动

### 4.3 题材与热点
- 政策题材：顶层设计驱动
- 事件题材：突发事件驱动
- 业绩题材：超预期驱动
- 热点生命周期：萌芽 → 扩散 → 高潮 → 退潮

### 4.4 市场情绪
- 情绪周期：乐观 → 兴奋 → 狂热 → 焦虑 → 恐慌
- 情绪指标：涨跌停数量、成交量、融资余额
- 情绪工具：昨日涨停表现、昨日炸板率
