# 短线量化分析模型
> 游资思维量化化 | 情绪周期 + 龙头博弈 + 主力行为 | 2026-04-27
> 与"老股民分析模板"完全独立，互不影响

---

## 框架总纲

```
短线量化的本质：做"资金合力 + 情绪溢价"

三大核心变量：
① 情绪周期（大环境） — 系统开关
② 题材强度（资金方向） — 选什么
③ 个股地位（是否龙头） — 买什么

完整策略流程：
Step1 判断情绪周期 → Step2 筛选Top题材 → Step3 识别龙头股
→ Step4 匹配买入模型 → Step5 执行风控规则
```

---

## 第一层：情绪周期识别（核心开关）

> **这是整个系统的"开关"**，决定是否操作以及用什么策略。

### 四阶段定义

| 阶段 | 量化标准 | 操作策略 |
|------|---------|---------|
| **冰点** | 跌停数 > 涨停数×1.5<br>连板高度 ≤ 2板<br>炸板率 > 40% | 空仓 or 极轻仓试错 |
| **修复** | 跌停减少<br>首板增加<br>有2~3板出现 | 轻仓低吸试错 |
| **主升** | 连板高度 ≥ 4板<br>涨停家数 > 80<br>龙头持续一字/换手 | **重仓做多（核心利润区）** |
| **退潮** | 龙头断板<br>高位股集体杀跌<br>炸板率上升 | 只做低位板 or 退出 |

### 量化指标计算

```
涨停家数：当日沪深涨停股票数量
跌停家数：当日沪深跌停股票数量
连板高度：当前最高连板数（以连板股为准）
炸板率 = 涨停炸板数 / 涨停总数 × 100%
首板家数：当日首次涨停股票数量（排除连板股）
```

### 情绪周期量化表

| 指标 | 冰点 | 修复 | 主升 | 退潮 |
|------|------|------|------|------|
| 涨停家数 | <30 | 30~60 | >80 | 40~80 |
| 跌停家数 | >40 | 10~30 | <10 | 20~50 |
| 涨停/跌停比 | <0.5 | 1~3 | >5 | 1~3 |
| 连板高度 | ≤2 | 2~3 | ≥4 | 断板 |
| 炸板率 | >40% | 25~40% | <25% | >35% |
| 首板增加 | 减少 | 增加 | 持续 | 减少 |

## 第二层：题材强度评分（决定做什么）

### 评分模型

```
题材强度 =
  0.4 × 涨停数量（权重最大）
+ 0.3 × 连板高度
+ 0.2 × 龙头封单强度
+ 0.1 × 板块成交额
```

### 筛选规则

```
① 只做 Top 3 题材（其他题材不参与）
② 强度 < 阈值 → 不参与（宁可错过）
③ 题材内部：只做龙头和次龙，不做跟风
```

### 题材分类操作表

| 题材类型 | 操作 | 说明 |
|---------|------|------|
| 主线（Top1） | 重仓参与 | 主升期核心利润来源 |
| 次主线（Top2~3） | 轻仓参与 | 轮动机会 |
| 过渡题材 | 不参与 | 无持续性 |
| 退潮题材 | 回避 | 资金已撤 |

## 第三层：龙头识别规则（核心Alpha）

> **短线的本质：不是选股，是选龙头**

### 龙头判定条件（必须满足3条以上）

```
条件1：连板高度 = 板块第一
条件2：首板时间最早（时间优先，越早封板越强）
条件3：封单金额最大（涨停封单量）
条件4：换手合理（非一字死板，有换手才健康）
```

### 龙头分类

| 类型 | 特征 | 操作 |
|------|------|------|
| **真龙** | 满足全部4条 | 重仓打板/回封 |
| **次龙** | 满足3条，题材Top3 | 轻仓参与，回调低吸 |
| **跟风** | 不满足龙头条件 | 不参与 |
| **杂毛** | 无板块效应 | 坚决回避 |

### 龙头生命周期

```
首板（发现）→ 2板（确认）→ 3板（主升）→ 4板+（加速）→ 断板（退潮）
          ↑                                    ↓
          ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ←
```

## 第四层：买入信号规则（3种核心模型）

### 模型1：打板（最强模式）

**核心逻辑**：在个股确认成为龙头瞬间买入

```
必要条件：
① 题材强度 Top 3
② 个股为龙头 or 次龙
③ 封板时成交量放大（量比 > 1.5x）
④ 封单金额 > 流通市值1%（封板强度）

买入点：封板确认瞬间（即时成交单）
```

**打板量化标准**

```
打板信号：
  涨停价出现大单成交（分时量峰）
  封单金额 / 流通市值 > 1%
  最后3分钟封板且封单稳定
```

**打板禁止条件（任意一条都禁止）**

```
❌ 情绪周期处于冰点 or 退潮
❌ 题材不在Top3
❌ 非龙头/次龙
❌ 封板时间 > 14:30（一板早，封板晚=弱）
❌ 炸板后无法回封（主力已撤）
```

### 模型2：龙头回封（二次加速）

**核心逻辑**：涨停打开后，再次封板 = 主力确认继续做多

```
必要条件：
① 首次封板后炸板（开板换手）
② 开板时间 < 30分钟（快速回封=强）
③ 开板后承接强（分时均价线上方运行）
④ 再次封板时量能健康（换手率 < 15%）

买入点：回封瞬间
```

**回封分类**

| 类型 | 强度 | 操作 |
|------|------|------|
| 早盘10:00前回封 | 最强 | 积极买入 |
| 午盘13:00~14:00回封 | 中强 | 稳健买入 |
| 尾盘14:30后回封 | 弱 | 谨慎或放弃 |

### 模型3：低吸（风险最低）

**核心逻辑**：龙头回调到关键支撑，分时恐慌低点买入

```
必要条件：
① 情绪周期处于主升期
② 个股为龙头（回调才值得低吸）
③ 回调幅度：首次回调（幅度小），二次回调（幅度大）
④ 支撑位：5日线 or 10日线
⑤ 量能：缩量（主力没出）

买入点：分时图恐慌砸出低点，企稳时
```

**低吸量化标准**

```
低吸信号：
  回调至MA5附近（价格 ≤ MA5×1.01）
  分时图出现恐慌性砸盘（快速下杀）
  砸盘后3分钟内拉回均价线上方
  量能萎缩（< 均量50%）
```

**低吸禁止条件**

```
❌ 情绪周期不在主升期
❌ 非龙头（跟风股低吸=接刀）
❌ 回调幅度 > 15%（可能是下跌中继）
❌ 跌破10日线（龙性已失）
```

## 第五层：卖出规则（必须机械化）

> **短线交易，卖出比买入更重要。无脑执行，不幻想。**

### 止盈规则

```
止盈1：+5%~+8% → 减仓1/3
止盈2：次日冲高不板 → 减仓2/3
止盈3：成本+15%以上 → 底仓持有，趋势不破不走
```

### 止损规则

```
止损1：-5% 必走（无条件）
止损2：跌破分时均价线 → 走（不幻想）
止损3：-3%出现放量下杀 → 立即走
```

### 情绪止损（最关键！）

```
卖出信号：
  龙头断板（当日无法涨停）→ 全撤
  高位炸板且无法回封 → 全撤
  跌停板开盘 → 开盘就走
  跟风股涨幅 < +3%且龙头回落 → 走
```

### 卖出决策树

```
持仓中：
  涨 > +8%?
    Y → 减仓1/3，剩下移动止损
    N ↓
  跌 < -5%?
    Y → 全走（无脑执行）
    N ↓
  龙头断板?
    Y → 全走
    N ↓
  跌破分时均价?
    Y → 走
    N → 持有
```

## 第六层：仓位管理（决定能否稳定盈利）

### 仓位量化标准

```
总仓位 = 市场情绪系数 × 题材系数 × 个股系数

市场情绪系数：
  主升期：0.8~1.0（重仓）
  修复期：0.4~0.6（轻仓）
  冰点期：0.0~0.2（空仓/极轻）
  退潮期：0.0~0.3（只做低位）

个股仓位上限：
  龙头：40%（最强个股可超配）
  次龙：20%
  跟风股：10%（尽量不做）
  杂毛：0%（不做）
```

### 仓位管理对照表

| 情绪周期 | 单票仓位 | 总仓位 | 策略 |
|---------|---------|-------|------|
| 冰点 | ≤10% | ≤20% | 空仓休息为主 |
| 修复 | 10~20% | 20~40% | 轻仓试错 |
| 主升 | 20~40% | ≥70% | 重仓龙头 |
| 退潮 | ≤10% | ≤30% | 只做低位板 |

### 仓位分配原则

```
① 单票上限：龙头40%，次龙20%，跟风10%
② 单日总票数：≤5只（专注）
③ 单日总仓位：情绪周期决定上限
④ 加仓原则：确认盈利后才加，不摊成本
```

## 完整策略流程图

```
Step 1：判断情绪周期
    ↓
[冰点?] → Y → 空仓/退出，不操作
    ↓ N
[退潮?] → Y → 只做低位板，极轻仓
    ↓ N
Step 2：筛选 Top 3 题材
    ↓
[题材强度达标?] → N → 退出
    ↓ Y
Step 3：识别龙头/次龙
    ↓
[有龙头?] → N → 退出（不做跟风）
    ↓ Y
Step 4：匹配买入模型
    ├── 打板条件满足 → 打板买入
    ├── 回封条件满足 → 回封买入
    └── 低吸条件满足 → 低吸买入
    ↓
Step 5：执行风控规则
    ├── 止盈：+5%~+8%分批出
    ├── 止损：-5%必走
    └── 情绪止损：龙头断板全撤
```

## 短线三大禁忌（必背）

```
❌ 禁忌1：把短线做成趋势
   → 短线是情绪博弈，趋势是方向交易，节奏完全不同
   → 死法：短线不止损，套成股东

❌ 禁忌2：不做情绪过滤
   → 冰点期/退潮期重仓操作
   → 死法：胜率腰斩，本金快速磨损

❌ 禁忌3：迷信技术指标
   → MACD/RSI 在短线几乎无用（延迟信号）
   → 短线只看：情绪/题材/龙头/量价/封单
```

---

## 进阶方向：主力行为融合

```
主力吸筹 → 低吸模型（修复期）
主力拉升 → 打板模型（主升期）
主力出货 → 风控模型（退潮期）

情绪周期是外壳
主力行为是内核
两者结合 = 完整短线体系
```
