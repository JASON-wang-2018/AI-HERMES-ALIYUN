## 二、游资共性模式

### 2.1 龙头股模式
```
选股标准：
1. 板块龙头，题材正宗
2. 涨停板首次出现（首板）
3. 成交金额大（流动性好）
4. 涨停时封板资金大
5. 题材催化剂明确
```

### 2.2 涨停板战法
- **首板**：板块启动第一只涨停
- **二板**：确认板块强度
- **三板**：龙头确立
- **四板+**：妖股模式，高风险

### 2.3 次新股战法
- 上市初期，无老庄
- 情绪博弈为主
- 关注开板后的二波机会

### 2.4 题材炒作周期
```
消息催化
    ↓
龙头首板（试探）
    ↓
板块扩散（2-3日）
    ↓
后排补涨（行情末端）
    ↓
退潮（龙头滞涨）
```

## 三、散户常见错误

### 3.1 亏损模式
| 错误 | 描述 | 心理根源 |
|------|------|---------|
| 追高 | 股价涨了很多才买 | 嫉妒/FOMO |
| 死抗 | 亏损死拿不止损 | 损失厌恶 |
| 小赚就跑 | 赚了5%就卖 | 恐惧 |
| 频繁交易 | 每天买卖 | 多动症/寻求刺激 |
| 满仓梭哈 | 一次全仓 | 过度自信 |
| 听消息 | 盲目跟庄 | 依赖他人 |
| 逆势死扛 | 下跌趋势加仓 | 锚定成本 |

### 3.2 心理偏差
- **损失厌恶**：亏1万的痛苦 > 赚1万的快乐
- **结果偏差**：用结果判断决策对错
- **后见之明**：事后觉得自己早知道
- **过度自信**：70%的司机认为技术高于平均
- **羊群效应**：别人买我也买
- **锚定效应**：死盯买入价
- **处置效应**：套了死拿，赚了就跑

## 四、游资猎散手法

### 4.1 拉高出货
```
Step 1: 建仓（低位慢慢买）
Step 2: 拉升（连续阳线吸引眼球）
Step 3: 出货（高位震荡派发）
Step 4: 砸盘（剩余筹码清仓）
```

### 4.2 洗盘陷阱
- 假突破：突破后快速回落
- 假破位：跌破支撑后快速拉回
- 震仓：大幅震荡洗出浮筹

### 4.3 消息配合
- 利好前建仓，利好公布后出货
- 利用研报、消息拉抬股价

### 4.4 席位联动
- 多个券商席位配合
- 龙虎榜席位协同作战

## 五、识别游资痕迹

### 5.1 龙虎榜
- 频繁上榜的营业部
- 买入席位集中度
- 卖出席位分散度

### 5.2 分时图特征
- 尾盘拉升（做收盘价）
- 早盘快速拉停
- 涨停炸板（出货迹象）

### 5.3 成交量特征
- 涨停次日量能异常
- 高位放量滞涨
- 下跌途中放量反弹

## 六、跟庄策略

### 6.1 跟庄原则
1. 确认主力在（低位放量）
2. 确认方向（上升趋势）
3. 设置止损（跌破关键位就走）
4. 严格仓位管理
5. 不抄底，只顺势

### 6.2 庄家弱点
- 资金量大，出货需要时间
- 需要配合消息面
- 受大盘环境影响
- 底部吸筹怕被对家发现
