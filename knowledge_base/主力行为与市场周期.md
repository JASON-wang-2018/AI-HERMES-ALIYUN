## 一、主力坐庄全流程

### 1.1 七步标准流程

```
┌─────────────────────────────────────────────────────────┐
│  建仓 → 吸筹 → 试盘 → 洗盘 → 拉升 → 砸盘 → 出货         │
└─────────────────────────────────────────────────────────┘
```

#### 第一步：建仓（压价收集）
- **目的**：悄悄买入大量廉价筹码
- **特征**：
  - 股价低位横盘，成交量极度萎靡
  - 分时图不活跃，经常出现心电图走势
  - 恐慌情绪蔓延，散户割肉
- **时间**：数周到数月不等
- **筹码**：通常收集流通盘的30-50%

#### 第二步：吸筹（低位整固）
- **目的**：进一步压低持仓成本
- **特征**：
  - 底部反复震荡，偶尔出现假突破
  - 利好不涨，利空反而上涨（主力控盘）
  - 筹码峰在低位聚集（吸筹峰）
- **时间**：1-3个月

#### 第三步：试盘（测试抛压）
- **目的**：测试上方的抛压和跟风盘
- **特征**：
  - 突然放量拉升后快速回落（冲高回落）
  - 或突然打压后快速拉起（挖坑测试）
  - 分时图振幅加大
- **结果**：根据试盘结果调整拉升计划

#### 第四步：洗盘（清理浮筹）
- **目的**：洗掉获利盘和不坚定筹码
- **特征**：
  - 快速回调但不会跌破重要支撑
  - 成交量萎缩（主力没出货）
  - 散户恐慌抛售，但股价很快收复
  - 典型形态：长阴短柱、倒锤头
- **时间**：1-4周

#### 第五步：拉升（脱离成本区）
- **目的**：快速拉高股价，脱离成本区
- **特征**：
  - 连续阳线，涨停板
  - 成交量持续放大
  - 分时图干净利落
  - 5日线陡峭
- **时间**：数周到数月

#### 第六步：砸盘（高位出货前奏）
- **目的**：制造恐慌，掩护出货
- **特征**：
  - 高位长阴线
  - 成交量异常放大
  - 跌停板出现
  - 多出现在尾盘砸盘

#### 第七步：出货（高位派发）
- **目的**：把筹码换成现金
- **特征**：
  - 高位震荡，涨少跌多
  - 利好频出但股价不涨
  - 成交量大但股价滞涨
  - 龙虎榜显示机构卖出
- **方式**：
  - 震荡出货：拉高出货
  - 打压出货：直接砸盘
  - 利好出货：借利好派发

## 二、K线背后的资金行为

### 2.1 建仓期K线特征
- **低量横盘**：量能极度萎缩，波动极小
- **假破位**：偶尔跌破重要支撑后又拉回
- **逆势抗跌**：大盘跌它不跌或跌得少
- **缩量新低**：不断缩量创出新低

### 2.2 拉升期K线特征
- **量增价涨**：量价配合完美
- **涨停基因**：经常出现涨停板
- **阶梯式上涨**：涨一段横盘，再涨一段
- **长阳短上影**：大阳线配合短上影线

### 2.3 出货期K线特征
- **放量滞涨**：成交量大但涨不动
- **高位长上影**：经常出现长上影线
- **分时图心电图**：上下反复震荡出货
- **涨少跌多**：整体呈现下降趋势

## 三、板块轮动规律

### 3.1 A股典型板块轮动
```
牛市初期：券商 → 房地产 → 基建
牛市中期：科技 → 消费 → 医药
牛市末期：低价股 → 垃圾股 → 消灭低价股
熊市初期：防御性板块（白酒/医药）→ 整体下跌
熊市中期：超跌反弹 → 热点题材
熊市末期：题材股 → 权重股补跌
```

### 3.2 政策驱动轮动
- 政策出台 → 政策受益板块启动
- 持续加码 → 板块持续炒作
- 政策收紧 → 板块退潮

### 3.3 资金驱动轮动
- 大资金建仓：先拉指标股
- 游资接力：题材股全面开花
- 散户追入：高位接盘

## 四、市场周期

### 4.1 牛熊周期
```
                    牛市
    ┌───────────────────────────────────────┐
    │  犹豫期 → 发展期 → 高潮期 → 转折期    │
    │     ↓                    ↓            │
    │   券商                      高位震荡   │
    │   金融                      利好频出   │
    │   龙头                      散户疯狂   │
    └───────────────────────────────────────┘
                    ↓
    ┌───────────────────────────────────────┐
    │  熊市                                 │
    │  初期暴跌 → 中期反弹 → 末期阴跌       │
    │     ↓            ↓          ↓        │
    │  杠杆出清   超跌反弹    信心崩溃     │
    └───────────────────────────────────────┘
```

### 4.2 情绪周期
| 阶段 | 特征 | 情绪 | 操作 |
|------|------|------|------|
| 绝望 | 大面积下跌，利好无效 | 恐慌 | 观望/定投 |
| 犹豫 | 涨跌互现，热点散乱 | 迷茫 | 轻仓试探 |
| 回暖 | 指数企稳，龙头股涨 | 谨慎乐观 | 逐步加仓 |
| 兴奋 | 连续上涨，赚钱效应 | 贪婪 | 持仓待涨 |
| 狂热 | 全民炒股，街头议股 | 极度贪婪 | 逐步离场 |

## 五、价格位置与筹码分析

### 5.1 价格位置判断
- **高位（>80%分位）**：累计涨幅大，风险积累
- **中位（40-80%分位）**：趋势延续或反转都有可能
- **低位（<20%分位）**：风险释放，可关注机会
- **历史高点附近**：套牢盘重，需要放量突破
- **历史低点附近**：支撑强，可观察止跌信号

### 5.2 筹码峰分析
- **低位密集峰**：主力建仓/吸筹区域
- **高位密集峰**：主力出货/散户接盘区域
- **筹码发散**：趋势延续
- **单峰集中**：变盘前兆

## 六、关键支撑压力位

### 6.1 支撑位
- 前期低点
- 均线支撑（20/60/120日线）
- 整数关口
- 跳空缺口
- 黄金分割位

### 6.2 压力位
- 前期高点
- 均线压力
- 前期成交密集区
- 黄金分割位
- 整数关口
