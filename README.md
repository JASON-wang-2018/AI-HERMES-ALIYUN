## 🎯 两套分析模型（完全独立）

| 模型 | 老股民分析模板 | 短线量化模型 |
|------|--------------|------------|
| 定位 | 波段/中线 | 超短/T+1 |
| 核心逻辑 | 趋势+主力行为 | 情绪周期+龙头博弈 |
| 指标权重 | MACD/RSI重要 | 技术指标几乎无用 |
| 卖出依据 | 均线/形态/止损 | 情绪止损机械化 |
| 盈利来源 | 趋势延续 | 情绪溢价 |
| 输出 | 个股综合报告（7步） | 每日选股清单+情绪周期 |

## 🚀 快速启动

### 激活环境
```bash
cd ~/stock_knowledge
source venv/bin/activate
```

### 采集数据
```bash
python3 scripts/data_fetch/stock_data_fetcher.py
```

### 运行分析
```bash
# 技术分析（老股民模板）
python3 scripts/analysis/stock_technical_analysis.py

# 趋势模型
python3 model_base/trend_model.py

# 情绪模型
python3 model_base/sentiment_model.py
```
