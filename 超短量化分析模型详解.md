# 超短量化分析模型 · 详解

> 版本：v1.0 | 日期：2026-05-02
> 作者：纳福（基于Jason的A股实战框架构建）
> 定位：A股T+1超短线量化分析，聚焦"今日买明日卖"的隔日短线逻辑

---

## 一、模型核心定位

### 1.1 解决什么问题

A股超短线交易的核心矛盾是：**市场每天都在波动，但只有极少数股票值得做。**

大多数短线交易者亏损的原因，不是技术不够，而是：
- **选股随缘** — 看到某只股涨了就追，不判断市场整体环境
- **重仓出击不看时机** — 退潮期满仓，亏钱效率最高
- **龙头跟风混淆** — 买了跟风股（🐍），跟龙头（🐉）走势天差地别

超短量化模型解决这三个问题，核心思路是：**在大机会来临时重拳出击，在市场退潮时全面防守。**

---

### 1.2 两套独立模型的关系

| 模型 | 时间周期 | 核心逻辑 | 适用场景 |
|------|---------|---------|---------|
| **超短量化模型** | T+1（今日买明日卖） | 情绪周期=系统开关，题材强度=方向，龙头=标的 | 每天收盘后分析，次日执行 |
| **老股民分析模板** | 波段/中线 | 10维度技术验证+100分综合评分 | 个股深度研究，仓位配置 |

两套模型**等级平行，互不替代**。同一个股票，两套模型可能给出截然不同的结论——这是正常的，因为它们的出发点和周期不同。

## 二、第一层：情绪周期识别（系统开关）

### 2.1 核心思想

> **情绪周期是超短线交易的"系统开关"：开关不开，操作就是逆水行舟。**

情绪周期反映的是**全市场短线资金的整体风险偏好**。牛市顶部区域，情绪高涨，即使选股平庸也容易赚钱；熊市底部区域，情绪冰点，即便选了最好的股票也容易被大盘拖累。

超短线交易依赖的是**对手盘** — 你明天卖给谁，取决于今天市场整体是否愿意接力。情绪周期就是判断"明天有没有人接盘"的工具。

### 2.2 四个核心指标

#### 指标一：涨停/跌停比（最核心）

$$涨停/跌停比 = \frac{涨停家数}{跌停家数}$$

**逻辑**：
- 涨停家数代表市场上最激进的那批资金的活跃程度
- 跌停家数代表亏钱效应的扩散程度
- 比值 > 3，说明市场上赚钱效应远大于亏钱效应
- 比值 < 1，说明亏钱效应开始扩散，连板被砸

**信号判断**：

| 比值区间 | 信号 | 含义 |
|---------|------|------|
| ≥ 3.0 | ✅ 极强 | 主升高潮，遍地涨停，积极做多 |
| 1.5 ~ 3.0 | ➕ 偏强 | 正常多头，可参与 |
| 1.0 ~ 1.5 | ➖ 偏弱 | 谨慎，轻仓 |
| < 1.0 | 🔴 危险 | 亏钱效应扩散，不做 |

#### 指标二：炸板率

$$炸板率 = \frac{炸板家数}{涨停家数 + 炸板家数} \times 100\%$$

**逻辑**：
- 炸板率反映的是"今天涨停的股票，明天还有多少愿意锁仓"
- 炸板率越高，说明今天打板的人大多数被套，明天市场接力意愿越低
- 炸板率高 = 打板策略当日亏损率高 = 次日新打板资金会犹豫

**信号判断**：

| 炸板率区间 | 信号 | 含义 |
|-----------|------|------|
| ≤ 15% | ✅ 极好 | 封板率高，锁仓意愿强 |
| 15% ~ 25% | ➕ 正常 | 可操作 |
| 25% ~ 40% | ⚠️ 偏高 | 打板谨慎 |
| > 40% | 🔴 危险 | 打板等于送钱 |

#### 指标三：最高连板数

**逻辑**：
- 最高连板数代表市场**最高赚钱高度**
- 5板股出现 = 市场有勇气打出绝对高度，说明短线资金极度活跃
- 没有连板股（最高1板）= 市场高度被压制，观望情绪浓厚

**信号判断**：

| 最高连板 | 信号 | 含义 |
|---------|------|------|
| ≥ 5板 | ✅ 超级龙头 | 极端多头 |
| 3~4板 | ➕ 正常 | 有高度，有赚钱效应 |
| 2板 | ➖ 偏弱 | 高度有限 |
| 1板（无连板） | 🔴 冰点 | 市场放弃做多 |

#### 指标四：连板股数量

**逻辑**：
- 连板股数量代表**中位龙头**的分布密度
- 连板股多 = 各板块都有资金在做，不是单一个股孤军奋战
- 连板股少 = 只有零星个股表演，无法形成板块联动效应

### 2.3 综合评分（100分制）

四个指标各自打分后加权，得到情绪周期总分：

| 指标 | 权重 | 满分 |
|------|------|------|
| 涨停/跌停比 | 40% | 40分 |
| 炸板率 | 30% | 30分 |
| 最高连板 | 20% | 20分 |
| 连板股数量 | 10% | 10分 |

### 2.4 阶段划分与仓位对应

| 阶段 | 分数区间 | 仓位建议 | 操作策略 |
|------|---------|---------|---------|
| 🟢 主升 | ≥ 75分 | 60~80% | 积极做多，精选真龙打板 |
| 🔵 修复 | 60~74分 | 30~50% | 轻仓试错，聚焦龙头 |
| ⚪ 震荡 | 40~59分 | 20~30% | 观望，快进快出 |
| 🟡 退潮 | 25~39分 | 0~10% | 极度防守，空仓为主 |
| 🔴 冰点 | < 25分 | 0% | 完全空仓，休息 |

**关键原则**：情绪周期决定仓位上限。即使某只股票看起来再好，在冰点期也不重仓——因为那是逆水行舟，成功率极低。

## 三、第二层：题材强度排行榜

### 3.1 核心思想

> **超短线赚钱的核心是"坐电梯"——跟随主力资金集中进攻的方向。**

A股的超短线收益来源于**板块效应**：同一题材的股票联动上涨，主力资金集团作战，形成板块内多股涨停的壮观场面。单独上涨的个股很难走远，但板块联动效应一旦形成，龙头股可以连续涨停。

题材强度分析解决的核心问题是：**今天市场上哪个方向的资金最集中？**

### 3.2 数据来源

从涨停池数据中提取：
- 涨停股的**所属行业**（聚合）
- 每只涨停股的**连板数、封单金额、成交额、换手率**

### 3.3 题材强度公式

$$题材强度分 = 0.4 \times 涨停数量分 + 0.3 \times 最高连板分 + 0.2 \times 封单强度分 + 0.1 \times 成交额分$$

**各分项说明**：

| 维度 | 权重 | 计算方式 | 逻辑 |
|------|------|---------|------|
| 涨停数量 | 40% | 该题材涨停股数 / 全市场最高涨停数 × 100 | 资金聚集规模的直接体现 |
| 最高连板 | 30% | 该题材最高连板数 / 全市场最高连板数 × 100 | 代表题材内部赚钱效应的绝对高度 |
| 封单合计 | 20% | 该题材所有涨停股封单之和 / 最大题材封单之和 × 100 | 次日潜在买盘强度的领先指标 |
| 成交额 | 10% | 该题材成交额归一化 | 流动性配合，衡量资金进出难度 |

**归一化方法**：每项指标各自除以当日的最大值，最大值题材得100分，其他按比例折算。

### 3.4 题材强度的实战含义

- **题材强度高（>80分）**：市场主力资金在这个方向形成了共识，可能持续2~5天，积极参与
- **题材强度中（50~80分）**：资金有一定关注，可轻仓试探
- **题材强度低（<50分）**：资金分散，市场无主线，不重仓

### 3.5 题材的生命周期

| 阶段 | 特征 | 操作 |
|------|------|------|
| 启动 | 板块内出现首板股，有1~2个涨停 | 识别题材，观察持续性 |
| 发酵 | 涨停扩散，3~5个涨停，板块效应形成 | 积极参与龙头 |
| 高潮 | 板块全面涨停，连板高度达到4板以上 | 注意撤退，不要追跟风 |
| 退潮 | 炸板率上升，连板股开始断板 | 立即止损，不抄底 |

## 四、第三层：龙头识别

### 4.1 核心思想

> **超短线只做龙头，不做跟风。龙头涨停一次顶跟风涨三天。**

什么是龙头？龙头是在某个题材中**最早启动、封板最坚决、连板最高**的那只股票。资金选择龙头是因为龙头有更好的流动性、更好的溢价效应、更好的出货机会。

### 4.2 龙头分类

| 类型 | 定义 | 操作价值 |
|------|------|---------|
| 🐉 真龙 | 满足4条龙头条件中的3条以上 | 积极参与 |
| 🐉 次龙 | 满足2条龙头条件 | 可低吸 |
| 🐍 跟风 | 满足0~1条龙头条件 | 不参与 |

**跟风股的特征**：跟板块龙头同时涨停，但封板时间晚、封单小、连板弱。跟风股在板块退潮时最先炸板，且回撤幅度远大于龙头。

### 4.3 龙头判定四条标准

#### 标准一：连板高度 = 板块第一

**逻辑**：连板高度代表市场对这只股票的价值共识。2板比1板强，3板比2板强。板块内连板最高的股票，就是板块内资金最认可的标的。

#### 标准二：首板时间最早（前5分钟）

**逻辑**：在同一题材中，最早涨停的股票是"点火者"，是主力资金最先发起进攻的信号。跟风股点火时间晚，通常在龙头封板后才拉升。

**量化方式**：比较首次封板时间，越早代表越先启动。

#### 标准三：封单金额最大

**逻辑**：封单是次日潜在买盘的"预约订单"。封单越大，说明主力资金锁仓意愿越强，愿意用真金白银表态。

**量化指标**：封单/流通市值 = 封单强度（>1%算强）

#### 标准四：换手率合理（0.5%~25%）

**逻辑**：
- 换手率 < 0.5%：说明股票筹码流动性太差，次日砸盘困难
- 换手率 > 25%：说明筹码换手过于充分，主力可能在出货
- 0.5%~25%：黄金区间，筹码换手充分但不过度

### 4.4 两种买入信号

#### 📌 信号一：打板（追涨停）

**触发条件（同时满足）**：
1. 连板数 ≥ 2 — 必须是连板股才有打板价值
2. 封单强度（封单/流通市值）≥ 1% — 封单不够大容易被砸
3. 换手率在合理区间（0.5%~25%）

**打板的核心逻辑**：
- 打板是在股票涨停的瞬间买入，本质上是"确认信号"后入场
- 打板成功的关键：次日要有溢价（即高开或连板），否则炸板当日亏损巨大
- 打板要求市场情绪好（第一层在主升阶段）、题材强度高（第二层有板块联动）

**打板失败的标准止损**：
- 买入后当日炸板收盘 → -5% 无条件止损
- 次日低开低走不反包 → 止损出局

#### 📥 信号二：低吸（回调买入）

**触发条件（同时满足）**：
1. 换手率 < 15% — 说明股票没有过度换手，主力仍在
2. 龙头类型为 🐉 真龙 或 🐉 次龙 — 不参与 🐍 跟风股

**低吸的核心逻辑**：
- 低吸是在股票回调时买入，属于"逆向交易"
- 低吸的目标是买在主力拉升前的洗盘低点
- 适用于龙头股第一次/第二次回调，不适用于连续回调的跟风股

**低吸失败的标准止损**：
- 买入后下跌 -3% → 预警
- 买入后下跌 -5% → 必须出局
- 持仓超过 3 个交易日未启动 → 强制离场

### 4.5 龙头与跟风的核心区别

| 特征 | 🐉 龙头 | 🐍 跟风 |
|------|--------|--------|
| 涨停时间 | 早（板块前5分钟） | 晚（龙头封板后） |
| 封单金额 | 大（封单强度≥1%） | 小 |
| 连板高度 | 高（板块第一） | 低或不连板 |
| 次日溢价 | 有（高开/连板） | 低或平开低走 |
| 回调幅度 | 小（通常<5%） | 大（可能跌停） |
| 板块退潮时 | 晚跌或不跌 | 第一个炸板 |

**实战口诀**：
> "宁在龙头上哭，不在跟风里笑"——买跟风股可能偶尔赚钱，但回撤大、胜率低，是散户亏钱的重灾区。

## 五、三层模型如何协同

### 5.1 决策流程

```
第一步：看情绪周期（系统开关）
    ↓ 分数 < 60分？
    → 是：轻仓或空仓，不往下走
    → 否：进入第二步

第二步：看题材强度（选方向）
    ↓ 有高强度题材（>80分）？
    → 有：进入第三步，在该题材中找龙头
    → 无：观望，不强行操作

第三步：龙头识别（选标的）
    ↓ 有真龙/次龙？
    → 有：打板 or 低吸，按信号执行
    → 无：不操作

同时：节前最后2个交易日强制降仓
```

### 5.2 三层权重分配

| 层级 | 决定内容 | 权重 |
|------|---------|------|
| 第一层 | **能不能做**（系统开关） | 40% |
| 第二层 | **做什么方向**（题材选择） | 35% |
| 第三层 | **做什么标的**（个股选择） | 25% |

### 5.3 典型决策案例

**案例一：情绪主升 + 题材强 + 有龙头**
- 条件：第一层85分🟢，第二层电力题材强度79分🥇，第三层华电能源4板🐉次龙
- 决策：积极参与低吸，仓位50%
- 逻辑：大盘好+主线明确+有明确标的

**案例二：情绪退潮 + 无题材 + 无龙头**
- 条件：第一层35分🟡，第二层无题材>80分，第三层000686不在涨停池
- 决策：空仓休息
- 逻辑：系统开关关闭，强行操作等于逆水行舟

## 六、风控体系

### 6.1 机械止损四条铁律

| 情况 | 止损规则 |
|------|---------|
| 打板失败 | 买入后当日炸板收盘，-5% 无条件止损 |
| 低吸失败 | 买入后下跌 -3% 预警，-5% 必须出局 |
| 持仓超时 | 超过 3 个交易日未启动，强制离场 |
| 板块冲高回落 | 板块当日冲高回落，隔日不反包，止损 |

### 6.2 操作禁区（触之必死）

| 禁区 | 原因 |
|------|------|
| 跟风股（🐍）不追 | 跟板块退潮一起炸板，回撤巨大 |
| 换手率>25%不接力 | 筹码过度分散，主力可能在出货 |
| 冰点/退潮期不打板 | 炸板率极高，打板等于送钱 |
| 节前最后1个交易日不满仓 | 假期消息面不确定性高 |

### 6.3 仓位管理原则

| 情绪阶段 | 仓位上限 | 单只标的仓位 |
|---------|---------|-------------|
| 🟢 主升（≥75） | 60~80% | 20~30%（可分散2~3只） |
| 🔵 修复（60~74） | 30~50% | 10~15%（轻仓试探） |
| ⚪ 震荡（40~59） | 20~30% | 5~10%（快进快出） |
| 🟡 退潮（25~39） | 0~10% | 0%（极轻仓或空仓） |
| 🔴 冰点（<25） | 0% | 空仓休息 |

## 七、数据来源与指标计算

### 7.1 主要数据源

| 数据 | 来源 | 用途 |
|------|------|------|
| 涨停池 | AKShare `stock_zt_pool_em` | 计算涨停家数、连板数、题材聚合 |
| 跌停池 | AKShare `stock_zt_pool_dtgc_em` | 计算跌停家数 |
| 炸板池 | AKShare `stock_zt_pool_zbgc_em` | 计算炸板率 |
| 个股实时行情 | AKShare `stock_zh_a_spot_em` | 个股换手率、封单、量比 |

### 7.2 核心指标计算公式

```python
# 涨停/跌停比
zt_dt_ratio = 涨停家数 / max(跌停家数, 1)

# 炸板率
zbg_rate = 炸板家数 / (涨停家数 + 炸板家数) * 100

# 题材强度分
theme_score = (
    0.4 * (涨停数 / max_zt_count) +
    0.3 * (最高连板 / max_lianban) +
    0.2 * (封单合计 / max_fengdan) +
    0.1 * (成交额 / max_vol)
) * 100

# 封单强度
fengdan_strength = 封单金额 / 流通市值 * 100

# 量比
vol_ratio = 当日成交量 / 5日均量
```

## 九、模型局限性与使用边界

### 9.1 模型的局限性

1. **T+1制度限制**：模型基于次日溢价逻辑，但T+1制度意味当天买的股票无法当天卖，盘中波动无法控制
2. **消息面无法量化**：政策利好、突发事件可能瞬间改变市场情绪，模型无法预测
3. **主力行为反量化**：主力资金可能故意制造假信号（如炸板洗盘），模型指标可能滞后
4. **小盘股失真**：流通市值<5亿的股票容易被庄家控盘，涨停/连板逻辑可能失效

### 9.2 使用边界

| 场景 | 适用性 |
|------|--------|
| 主板股票 | ✅ 完全适用 |
| 创业板（30cm） | ⚠️ 涨幅更大但炸板率也更高，仓位需降 |
| 科创板 | ⚠️ 同上 |
| ST股/退市股 | ❌ 不适用，模型逻辑不成立 |
| 新股上市首日 | ❌ 不适用，无连板逻辑 |
| 北交所 | ❌ 流动性不足，模型指标可能失真 |

## 十、节前特殊处理

### 10.1 为什么节前要特殊处理

A股有"假期效应"：
- 五一假期（5天）、国庆假期（7天）、春节假期（7天）期间，外盘照常运行，消息面不确定性大
- 节前最后1~2个交易日，主力资金通常选择避险减仓，市场情绪容易变差
- 节后第一个交易日，如果外围市场出现重大变化，A股可能大幅低开

### 10.2 节前操作规则

| 节点 | 规则 |
|------|------|
| 节前最后2个交易日 | 仓位降到30%以下，不新开仓 |
| 节前最后1个交易日 | 收盘前清仓或≤10%轻仓过节 |
| 节后第一个交易日 | 视外围情况决定是否加仓 |

**核心原则**：假期越长，风险越大。五一/国庆假期持仓过夜，是用5~7天的未知风险去博1~2天的潜在收益，不划算。
